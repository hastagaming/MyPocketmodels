# MyPocketModels 🤖

**MyPocketModels** adalah aplikasi asisten AI berbasis Android yang memungkinkan Anda menjalankan model bahasa besar (LLM) langsung dari saku. Aplikasi ini mendukung integrasi dengan **Ollama** (untuk penggunaan lokal/server) dan **Hugging Face Inference API**.

## ✨ Fitur Utama
- **Dual Engine:** Pilih antara Ollama atau Hugging Face (Llama-3, Mistral, dll).
- **Persistent Memory:** Percakapan disimpan secara otomatis menggunakan Room Database.
- **Modern UI:** Antarmuka chat bersih menggunakan Jetpack Compose.
- **Secure:** Sistem penyembunyian API Token menggunakan BuildConfig dan GitHub Secrets.
